# Nelfusion

A UCI-compliant chess engine written in Rust.

## Status

 **Work in progress.** Currently in active development.

-  Board representation, move generation, make/unmake (Currently perft is work in progress)
-  Search, evaluation, and UCI protocol — not yet implemented

## Building

```bash
cargo build --release
```

## Testing

```bash
cargo test
```
